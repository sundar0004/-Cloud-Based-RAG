# Six-Month Execution Timeline

## Month 1
- Literature review and problem framing.
- Environment setup and tooling.
- Initial dataset identification.

## Month 2
- Build ingestion pipeline.
- Chunking and embedding generation.
- FAISS indexing and retrieval validation.

## Month 3
- Build baseline LLM and RAG API.
- Prompt engineering for grounded responses.
- Internal demo v1.

## Month 4
- Implement experiment harness and metrics.
- Pilot experiment and parameter tuning.
- Mid-project review with supervisor.

## Month 5
- Docker hardening and cloud deployment.
- End-to-end testing and reliability checks.
- Run final experiment batches.

## Month 6
- Analyze results and discuss findings.
- Final dissertation writing and revisions.
- Viva slides, demo script, and mock defense.
